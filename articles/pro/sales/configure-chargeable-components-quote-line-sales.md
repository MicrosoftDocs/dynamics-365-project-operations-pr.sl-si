# <a name="configure-the-chargeable-components-of-a-quote-line---lite"></a><span data-ttu-id="3403a-103">Konfiguriranje komponent podrobnosti ponudbe, ki se zaračunajo – poenostavljena različica</span><span class="sxs-lookup"><span data-stu-id="3403a-103">Configure the chargeable components of a quote line - lite</span></span>

<span data-ttu-id="3403a-105">Vrstica projektne pogodbe je sestavljena iz pojmov *vključenih* komponent in komponent, *ki se zaračunajo*.</span><span class="sxs-lookup"><span data-stu-id="3403a-105">A project-based quote line has the concept of *included* components and *chargeable* components.</span></span>

<span data-ttu-id="3403a-106">Za vključene komponente veljajo naslednji elementi:</span><span class="sxs-lookup"><span data-stu-id="3403a-106">Included components are subject to:</span></span>

  - <span data-ttu-id="3403a-107">Način zaračunavanja in razdelitve kupcev</span><span class="sxs-lookup"><span data-stu-id="3403a-107">Billing method and customer splits</span></span>

<span data-ttu-id="3403a-110">Podmnožico vključenih komponent je v polju **Vrsta obračunavanja** mogoče označiti kot zaračunljive.</span><span class="sxs-lookup"><span data-stu-id="3403a-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="3403a-111">Polje **Vrsta obračunavanja** predstavlja nabor možnosti, ki v okviru vrstice ponudbe omogoča naslednje vrednosti:</span><span class="sxs-lookup"><span data-stu-id="3403a-111">The **Billing Type** field is an option-set that allows the following values in the context of a quote line:</span></span>

  - <span data-ttu-id="3403a-112">Se zaračuna</span><span class="sxs-lookup"><span data-stu-id="3403a-112">Chargeable</span></span>
  - <span data-ttu-id="3403a-113">Se ne zaračuna</span><span class="sxs-lookup"><span data-stu-id="3403a-113">Non-chargeable</span></span>

<span data-ttu-id="3403a-114">Komponente, ki se zaračunajo, je mogoče določiti za opravila, vloge in kategorije transakcij.</span><span class="sxs-lookup"><span data-stu-id="3403a-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="3403a-115">Možnost zaračunavanja je določena v opravilih za posamezno vrstico ponudbe in velja za vse razrede transakcij, vključene v to vrstico.</span><span class="sxs-lookup"><span data-stu-id="3403a-115">Chargeability is defined on the tasks for a quote line and applies to all transaction classes included on that line.</span></span> <span data-ttu-id="3403a-116">Če je polje **Vključi opravila** prazno ali nastavljeno na možnost **Ves projekt**, zavihek **Opravila, ki se zaračunajo** ni na voljo.</span><span class="sxs-lookup"><span data-stu-id="3403a-116">If the **Include Tasks** field is set to **Entire project** or left blank, the **Chargeable Tasks** tab isn't available.</span></span>

<span data-ttu-id="3403a-117">Možnost zaračunavanja je določena v vlogah za podrobnost projektne pogodbe in velja samo za razred transakcije **Čas**.</span><span class="sxs-lookup"><span data-stu-id="3403a-117">Chargeability is defined on roles for a quote line and only applies to the **Time** transaction class.</span></span> <span data-ttu-id="3403a-118">Če je v vrstici ponudbe polje **Vključi čas** prazno ali nastavljeno na možnost **Ne**, zavihek **Vloge, ki se zaračunajo** ni na voljo.</span><span class="sxs-lookup"><span data-stu-id="3403a-118">If the field, **Include Time** is set to **No** on a project quote line, the **Chargeable Roles** tab isn't available.</span></span>

<span data-ttu-id="3403a-119">Plačljivost je določena v kategorijah transakcije za podrobnost projektne pogodbe in velja samo za razred transakcije **Strošek**.</span><span class="sxs-lookup"><span data-stu-id="3403a-119">Chargeability is defined on transaction categories for a  quote line and only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="3403a-120">Če je v vrstici ponudbe polje **Vključi čas** prazno ali nastavljeno na možnost **Ne**, zavihek **Kategorije, ki se zaračunajo** ni na voljo.</span><span class="sxs-lookup"><span data-stu-id="3403a-120">If the field, **Include Expenses** is set to **No** on a project quote line, the **Chargeable Categories** tab isn't available.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="3403a-121">Posodabljanje projektnega opravila kot zaračunljivega ali nezaračunljivega</span><span class="sxs-lookup"><span data-stu-id="3403a-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="3403a-122">Projektna naloga je lahko v posamezni vrstici ponudbe, ki temelji na projektu, zaračunljiva ali nezaračunljiva, kar omogoča naslednje nastavitve:</span><span class="sxs-lookup"><span data-stu-id="3403a-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>

<span data-ttu-id="3403a-123">Če vrstica ponudbe, ki temelji na projektu, vključuje možnost **Čas** in opravilo **T1**, je opravilo povezano s ponudbo kot zaračunljivo.</span><span class="sxs-lookup"><span data-stu-id="3403a-123">If a project-based quote line includes **Time** and the task **T1**, the task is associated to the quote line as chargeable.</span></span> <span data-ttu-id="3403a-124">Če obstaja druga vrstica ponudbe, ki vključuje možnost **Stroški**, lahko opravilo **T1** v vrstici ponudbe povežete kot nezaračunljivo.</span><span class="sxs-lookup"><span data-stu-id="3403a-124">If there is a second quote line that includes **Expenses**, you can associate the **T1** task on the quote line as non-chargeable.</span></span> <span data-ttu-id="3403a-125">Rezultat tega je, da se ves čas, zabeležen za opravilo, zaračuna, vsi zabeleženi stroški pa se ne zaračunajo.</span><span class="sxs-lookup"><span data-stu-id="3403a-125">The result is that all time recorded on the task is chargeable and all expenses recorded on the task are non-chargeable.</span></span>

<span data-ttu-id="3403a-126">Vrsta obračuna za opravilo je lahko konfigurirana na zavihku **Opravila, ki se zaračunajo** podrobnosti ponudbe, ki temelji na projektih, s posodobitvijo polja **Vrsta obračunavanja** v podmreži **Opravila vrstice pogodbe**.</span><span class="sxs-lookup"><span data-stu-id="3403a-126">A task's billing type can be configured on the **Chargeable Tasks** tab of a project-based quote line by updating the **Billing Type** field on the **Quote Line Tasks** subgrid.</span></span> <span data-ttu-id="3403a-127">Druga možnost je, da posodobite vrsto obračunavanja za opravilo projekta v polju **Vrsta obračunavanja** v podmreži v nastavitvah obračunavanja opravil projekta, ki prikazuje podrobnosti ponudb, povezane z opravilom.</span><span class="sxs-lookup"><span data-stu-id="3403a-127">Alternatively, you can update the billing type for a project task in the **Billing Type** field on the subgrid on the task billing setup of a project that shows the quote lines associated to a task.</span></span>

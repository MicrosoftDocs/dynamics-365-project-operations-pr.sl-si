  - <span data-ttu-id="45bbf-113">Se ne zaračuna</span><span class="sxs-lookup"><span data-stu-id="45bbf-113">Non-chargeable</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="45bbf-121">Posodabljanje projektnega opravila kot zaračunljivega ali nezaračunljivega</span><span class="sxs-lookup"><span data-stu-id="45bbf-121">Update a project task to be chargeable or non-chargeable</span></span>
